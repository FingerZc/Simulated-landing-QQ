QQ批量登陆利器

　　QQ批量登陆利器主要为囤积大量QQ号的盆友一次性批量登陆QQ(主要作用:防止号码被腾讯回收;升级)的工具, 该工具一共分为密码登陆、SID登陆两种方式.
　　本工具的登陆原理是模拟登陆3GQQ, 为做出任何盗取号码的后台, 请放心使用.
　　如果你是PHP的学习者, 可以访问 http://FingerArt.me(指尖上的艺术) 查看该工具的核心源码.

密码登陆

　　密码登陆方式即,需要填写你的QQ号及密码;
　　在该方式下可以选择自动生成SID, 获取到号码的SID后即可使用SID的登陆方式了, 使用SID登陆更加安全;
　　你可以勾选24小时全天登陆, 每30分钟(3GQQ自动在线30分钟)会自动登录一次, 不能关闭窗口, 否则不会操作成功.
格式
QQ1----密码1
QQ2----密码2
QQ3----密码3 例如:
1234567890----password
1234567890----password
1234567890----password

SID登陆

　　SID登陆方式即,只需要填写你的QQ号及SID;
　　你可以勾选24小时全天登陆, 每30分钟(3GQQ自动在线30分钟)会自动登录一次, 不能关闭窗口, 否则不会操作成功;
　　SID登陆无须输入验证码即可直接登陆成功, 相比前一种方式是安全、快捷的, 需要注意的是SID的有效期是30天.
格式
QQ1----SID1
QQ2----SID2
QQ3----SID3 例如:
1234567890----AZ2791dW7rJK10knRPO86L7d
1234567890----AZ2791dW7rJK10knRPO86L7d
1234567890----AZ2791dW7rJK10knRPO86L7d

什么是SID

　　SID来自登陆3GQQ后的网址中:http://q32.3g.qq.com/g/s?aid=nqqchatMain&sid=AZ2791Yq7rJK90knvPO86L7d&myqq=10865012,SID每30天会被重新分配一个, 所以SID的有效期是30天; 因为腾讯为了兼容不支持COOKIE的手机, 所以会在地址上带上SID来识别用户.
